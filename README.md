# Location Validation

An asynchronous full-stack digital menu and ordering application that leverages real-time mobile GPS tracking to dynamically authorize user transaction features based on administrative property boundaries.

##  System Architecture Features
* **Geospatial Processing Engine:** Custom-engineered Ray-Casting (Point-in-Polygon) geometric engine running on FastAPI.
* **Responsive UI Interface:** Client-facing web layer styled with matching global brand hex codes, interfacing natively with the HTML5 Geolocation streaming hardware API.
* **Audit Trail Ledger:** Persistent relational data mapping engine utilizing MySQL to capture live operational logging telemetry.
